# Portfolio_Shop
간단한 쇼핑몰 프로젝트

개발 환경 : java 1.8.0, Spring Legacy, MySQL 8.0.21, apache-tomcat-8.5.57, window 10

개발 기간 : 2021.03.10 ~ 

개발 인원 : 1명

요구사항 정리

회원이 상품을 구매한다<br>
회원이 상품을 카트에 담을 수 있다
회원이 요구하는 배송지에 배송을 한다
회원을 0 또는 1곳 이상의 배송지를 등록할 수 있다
회원은 상품후기를 달 수 있다

- 회원
	  회원가입
    회원정보수정(아이디 / 패스워드 변경)
    회원탈퇴
    장바구니 담기
    주문하기
    주문취소
    환불
    배송지 설정
    후기 작성(상품 주문시)
    관리자에게 문의	

- 관리자
    상품 등록
    상품 수정
    상품 삭제 
    상품 배송과정 
  	매출 관리
